# Replit Prompt: StayBnB / RoomTrev — Full-Stack Room Booking Platform

## Project Overview

Build a full-stack web application called **StayBnB (RoomTrev)** — a platform that connects PG/hotel owners and tenants to simplify finding and booking PG accommodations. Think Airbnb but focused on PG rooms, private rooms, and shared accommodations.

---

## Tech Stack

- **Frontend:** React + React Router + Tailwind CSS + React Query
- **Backend:** Java 17 + Spring Boot 3.x
- **ORM:** Spring Data JPA + Hibernate
- **Database:** PostgreSQL (or H2 for local dev/Replit)
- **Auth:** Spring Security + JWT (jjwt library)
- **Password Hashing:** BCryptPasswordEncoder (built into Spring Security)
- **Payments:** Mock Stripe integration (plain Java service, no SDK needed)
- **Image Uploads:** Cloudinary Java SDK (or mock with local storage for Replit)
- **Build Tool:** Maven (`pom.xml`)
- **API Style:** RESTful JSON with `@RestController`

---

## Roles / User Types

1. **GUEST** — Can search, book rooms, leave reviews
2. **HOST** — Can list, edit, delete rooms; manage bookings
3. **ADMIN** — Full control over users, rooms, bookings

## Entity Definitions (JPA)

### User.java
```java
@Entity @Table(name = "users")
public class User {
    @Id @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;
    private String name;
    @Column(unique = true) private String email;
    private String password;           // BCrypt hashed
    @Enumerated(EnumType.STRING)
    private Role role;                 // USER, HOST, ADMIN
    private String phone;
    private String address;
    private String profilePicture;
    private boolean isVerified;
    private LocalDateTime createdAt;
}
```

### Room.java
```java
@Entity @Table(name = "rooms")
public class Room {
    @Id @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;
    @ManyToOne private User host;
    private String title;
    private String description;
    @Enumerated(EnumType.STRING) private RoomType type;  // ENTIRE, PRIVATE, SHARED
    private Double pricePerNight;
    private String location;
    private String city;
    @ElementCollection private List<String> amenities;
    @ElementCollection private List<String> images;
    private boolean isAvailable;
    private Double avgRating;
    private Integer reviewCount;
    private LocalDateTime createdAt;
}
```

### Booking.java
```java
@Entity @Table(name = "bookings")
public class Booking {
    @Id @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;
    @ManyToOne private Room room;
    @ManyToOne private User guest;
    private LocalDate checkIn;
    private LocalDate checkOut;
    private Integer guestCount;
    private Double totalPrice;
    @Enumerated(EnumType.STRING) private BookingStatus status;  // PENDING, CONFIRMED, CANCELLED
    private String paymentStatus;
    private LocalDateTime createdAt;
}
```

### Review.java
```java
@Entity @Table(name = "reviews")
public class Review {
    @Id @GeneratedValue(strategy = GenerationType.IDENTITY)
    private Long id;
    @ManyToOne private Room room;
    @ManyToOne private User user;
    @Min(1) @Max(5) private Integer rating;
    private String comment;
    private LocalDateTime createdAt;
}
```

## REST API Endpoints (Spring Controllers)

### AuthController — `/api/auth`
| Method | Path | Access | Description |
|--------|------|--------|-------------|
| POST | `/register` | Public | Register: name, email, password, role |
| POST | `/login` | Public | Returns `{ token, user }` |
| GET | `/me` | Authenticated | Get current user from JWT |

### RoomController — `/api/rooms`
| Method | Path | Access | Description |
|--------|------|--------|-------------|
| GET | `/` | Public | List rooms with filters: `?city=&type=&minPrice=&maxPrice=&keyword=` |
| GET | `/{id}` | Public | Room detail with reviews |
| POST | `/` | HOST | Create room |
| PUT | `/{id}` | HOST (owner) | Update room |
| DELETE | `/{id}` | HOST (owner) | Delete room |
| POST | `/{id}/images` | HOST | Upload images |

### BookingController — `/api/bookings`
| Method | Path | Access | Description |
|--------|------|--------|-------------|
| POST | `/` | USER | Create booking (conflict check + price calc) |
| GET | `/my` | USER | Current user's bookings |
| GET | `/host` | HOST | All bookings for host's rooms |
| GET | `/{id}` | Authenticated | Booking detail |
| PUT | `/{id}/cancel` | USER (owner) | Cancel + mock refund |

### ReviewController — `/api/rooms/{roomId}/reviews`
| Method | Path | Access | Description |
|--------|------|--------|-------------|
| POST | `/` | USER | Add review (only if booking completed) |
| GET | `/` | Public | All reviews for a room |
| DELETE | `/api/reviews/{id}` | ADMIN or author | Delete review |

### AdminController — `/api/admin`
| Method | Path | Access | Description |
|--------|------|--------|-------------|
| GET | `/users` | ADMIN | All users |
| DELETE | `/users/{id}` | ADMIN | Delete user |
| GET | `/bookings` | ADMIN | All bookings |
| PUT | `/bookings/{id}/status` | ADMIN | Update booking status |

## Security Configuration (Spring Security)

```java
// SecurityConfig.java — configure filter chain
http
  .csrf(csrf -> csrf.disable())
  .authorizeHttpRequests(auth -> auth
    .requestMatchers("/api/auth/**").permitAll()
    .requestMatchers(HttpMethod.GET, "/api/rooms/**").permitAll()
    .requestMatchers("/api/admin/**").hasRole("ADMIN")
    .requestMatchers(HttpMethod.POST, "/api/rooms").hasRole("HOST")
    .anyRequest().authenticated()
  )
  .addFilterBefore(jwtAuthFilter, UsernamePasswordAuthenticationFilter.class);
```

JWT filter should:
1. Extract `Authorization: Bearer <token>` header
2. Validate token using `JwtUtil`
3. Load user via `UserDetailsService`
4. Set `SecurityContextHolder` authentication

## Key Business Logic (Service Layer)

### 1. Double-Booking Prevention (BookingRepository)
```java
// Custom JPQL query to detect overlapping bookings
@Query("""
  SELECT COUNT(b) FROM Booking b
  WHERE b.room.id = :roomId
  AND b.status = 'CONFIRMED'
  AND b.checkIn < :checkOut
  AND b.checkOut > :checkIn
""")
long countOverlappingBookings(Long roomId, LocalDate checkIn, LocalDate checkOut);
```
Throw `ResponseStatusException(HttpStatus.CONFLICT)` if count > 0.

### 2. Price Calculation (BookingService)
```java
long nights = ChronoUnit.DAYS.between(checkIn, checkOut);
double total = room.getPricePerNight() * nights * guestCount;
booking.setTotalPrice(total);
```

### 3. Rating Recalculation (ReviewService)
After saving a review:
```java
Double avg = reviewRepository.findAverageRatingByRoomId(roomId);
int count = reviewRepository.countByRoomId(roomId);
room.setAvgRating(avg);
room.setReviewCount(count);
roomRepository.save(room);
```

### 4. Mock Payment (PaymentService)

## Frontend (React) — API Layer

All Axios calls should point to `http://localhost:8080/api`. Attach JWT token to every protected request:

```js
// api/axiosInstance.js
import axios from 'axios';

const api = axios.create({ baseURL: 'http://localhost:8080/api' });

api.interceptors.request.use(config => {
  const token = localStorage.getItem('token');
  if (token) config.headers.Authorization = `Bearer ${token}`;
  return config;
});

export default api;
```
